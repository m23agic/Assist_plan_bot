# Telegram Reminder Bot for Vercel

Telegram-бот, который принимает напоминание и время, а потом отправляет сообщение в Telegram. Проект теперь подготовлен под Vercel: входящие сообщения идут через webhook, а отправка просроченных напоминаний вынесена в отдельный HTTP-маршрут.

## Что умеет

- работать через кнопки внизу чата
- создавать напоминание на конкретную дату и время
- создавать напоминание на сегодня или завтра по короткому времени
- показывать список активных напоминаний
- удалять напоминание по ID
- хранить данные в Vercel KV
- локально работать через JSON-файлы в папке `data`

## Команды

```text
/start
/help
/remind 2026-03-28 14:30 Купить молоко
/remind 18:45 Позвонить другу
/list
/delete 1
```

Или просто пользоваться кнопками:

- `Создать напоминание`
- `Список напоминаний`
- `Удалить напоминание`

Если время в формате `HH:MM` уже прошло, бот автоматически поставит напоминание на следующий день.

## Локальный запуск

1. Установи Node.js 18+.
2. Создай бота через [@BotFather](https://t.me/BotFather) и получи токен.
3. Скопируй `.env.example` в `.env`.
4. Вставь токен в `TELEGRAM_BOT_TOKEN`.
5. При необходимости поменяй `TIMEZONE`.
6. Установи зависимости:

```bash
npm.cmd install
```

7. Запусти бота:

```bash
npm.cmd start
```

Локально бот использует long polling и хранит данные в `data/reminders.json` и `data/chat-states.json`.

## Деплой на Vercel

1. Залей проект в GitHub.
2. Импортируй репозиторий в Vercel.
3. В Vercel добавь Storage `KV`.
4. Добавь переменные окружения:

```text
TELEGRAM_BOT_TOKEN=...
TIMEZONE=Asia/Yekaterinburg
APP_URL=https://your-project.vercel.app
TELEGRAM_WEBHOOK_SECRET=любой_секрет
DISPATCH_SECRET=другой_секрет
```

5. После деплоя зарегистрируй webhook:

```bash
npm.cmd run register-webhook
```

Webhook будет смотреть в `https://your-project.vercel.app/api/telegram`.

## Отправка напоминаний на Vercel

Для отправки просроченных напоминаний используется маршрут:

```text
GET /api/dispatch?secret=YOUR_DISPATCH_SECRET
```

Его можно вызывать из внешнего cron-сервиса.

## Важно про бесплатный Vercel

- Telegram webhook на Vercel работает нормально.
- Сам Vercel не даёт нам постоянный фоновый процесс как на обычном сервере.
- Поэтому точность напоминаний зависит от того, как часто кто-то вызывает `/api/dispatch`.
- На практике для бесплатного варианта обычно используют внешний scheduler, например cron-job.org или другой ping-сервис.
